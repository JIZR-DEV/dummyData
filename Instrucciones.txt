# dummyData
Dummy Data java y sql


#Instrucciones de Uso
1. Iniciar la Aplicación
Ejecuta la aplicación. Al iniciar, se abrirá la ventana principal titulada "Generador de Datos Ficticios".
La interfaz está dividida en varias partes: un panel superior para ingresar datos, un área de texto central para visualizar los registros generados, y botones para interactuar con la aplicación.
2. Configurar el Número de Registros
En el panel superior, encontrarás un campo etiquetado como "Número de Registros:". Aquí debes ingresar el número de registros ficticios que deseas generar.
El número debe ser un valor entero positivo. La aplicación generalmente permite un rango desde 1 hasta 50,000 registros para mantener un rendimiento óptimo.
3. Generar los Datos
Una vez hayas ingresado el número deseado de registros, haz clic en el botón "Generar".
Al hacer clic en este botón, la aplicación procesará tu solicitud y generará los registros ficticios, que incluyen matrícula, apellidos, nombre, correo electrónico y fecha de nacimiento.
Los datos generados aparecerán en el área de texto grande que ocupa la mayor parte de la interfaz, permitiéndote revisar los detalles de cada registro.
4. Exportar Datos a SQL
Si deseas exportar los datos generados a un archivo SQL, puedes hacer clic en el botón "Exportar a SQL".
Al hacer clic en este botón, se abrirá un diálogo para seleccionar la ubicación y el nombre del archivo donde deseas guardar los datos.
Elige la ubicación deseada y confirma. La aplicación generará un archivo SQL con los comandos para insertar los datos en una base de datos, informándote mediante un mensaje si la exportación fue exitosa o si hubo un error.
